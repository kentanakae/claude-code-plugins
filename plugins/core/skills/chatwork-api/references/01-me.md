# ユーザー・アカウント

## GET /me

自分自身の情報を取得する。

**認証スコープ**: `users.all:read`, `users.profile.me:read`

**パラメータ**: なし

**レスポンス (200)**:

| フィールド | 型 | 説明 |
|-----------|------|------|
| account_id | integer | アカウントID |
| room_id | integer | ルームID |
| name | string | 表示名 |
| chatwork_id | string | Chatwork ID |
| organization_id | integer | 組織ID |
| organization_name | string | 組織名 |
| department | string | 部署 |
| title | string | 役職 |
| url | string | WebサイトURL |
| introduction | string | 自己紹介 |
| mail | string | メールアドレス |
| tel_organization | string | 会社電話番号 |
| tel_extension | string | 内線番号 |
| tel_mobile | string | 携帯電話番号 |
| skype | string | Skype ID |
| facebook | string | Facebook ID |
| twitter | string | Twitter ID |
| avatar_image_url | string | アバター画像URL |
| login_mail | string | 登録メールアドレス |

```json
{
  "account_id": 123,
  "room_id": 322,
  "name": "John Smith",
  "chatwork_id": "tarochatworkid",
  "organization_id": 101,
  "organization_name": "Hello Company",
  "department": "Marketing",
  "title": "CMO",
  "url": "http://mycompany.example.com",
  "introduction": "Self Introduction",
  "mail": "taro@example.com",
  "tel_organization": "XXX-XXXX-XXXX",
  "tel_extension": "YYY-YYYY-YYYY",
  "tel_mobile": "ZZZ-ZZZZ-ZZZZ",
  "skype": "myskype_id",
  "facebook": "myfacebook_id",
  "twitter": "mytwitter_id",
  "avatar_image_url": "https://example.com/abc.png",
  "login_mail": "account@example.com"
}
```

## GET /my/status

自分の未読数・メンション数・タスク数を取得する。

**認証スコープ**: `users.all:read`, `users.status.me:read`

**パラメータ**: なし

**レスポンス (200)**:

| フィールド | 型 | 説明 |
|-----------|------|------|
| unread_room_num | integer | 未読のあるチャット数 |
| mention_room_num | integer | メンションのあるチャット数 |
| mytask_room_num | integer | 自分のタスクのあるチャット数 |
| unread_num | integer | 未読メッセージ総数 |
| mention_num | integer | メンション総数 |
| mytask_num | integer | 自分のタスク総数 |

```json
{
  "unread_room_num": 2,
  "mention_room_num": 1,
  "mytask_room_num": 3,
  "unread_num": 12,
  "mention_num": 1,
  "mytask_num": 8
}
```

## GET /my/tasks

自分のタスク一覧を取得する（最大100件）。

**認証スコープ**: `users.all:read`, `users.tasks.me:read`

**パラメータ**:

| パラメータ | 型 | 必須 | 説明 |
|-----------|------|------|------|
| assigned_by_account_id | integer | いいえ | タスク依頼者のアカウントIDでフィルタ |
| status | string | いいえ | `open`（未完了）または `done`（完了）でフィルタ |

**レスポンス (200)**: タスクオブジェクトの配列

| フィールド | 型 | 説明 |
|-----------|------|------|
| task_id | integer | タスクID |
| room.room_id | integer | ルームID |
| room.name | string | チャット名 |
| room.icon_path | string | チャットアイコンURL |
| assigned_by_account.account_id | integer | 依頼者のアカウントID |
| assigned_by_account.name | string | 依頼者の表示名 |
| assigned_by_account.avatar_image_url | string | 依頼者のアバターURL |
| message_id | string | 関連メッセージID |
| body | string | タスク内容 |
| limit_time | integer | 期限（Unixタイムスタンプ） |
| status | string | `open` または `done` |
| limit_type | string | `none`, `date`, `time` |

```json
[
  {
    "task_id": 3,
    "room": {
      "room_id": 5,
      "name": "Group Chat Name",
      "icon_path": "https://example.com/ico_group.png"
    },
    "assigned_by_account": {
      "account_id": 456,
      "name": "Anna",
      "avatar_image_url": "https://example.com/def.png"
    },
    "message_id": "13",
    "body": "buy milk",
    "limit_time": 1384354799,
    "status": "open",
    "limit_type": "date"
  }
]
```
